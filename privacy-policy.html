📚 Politique de Confidentialité - Inkavi

Dernière mise à jour : 27 février 2026

1. Introduction

Inkavi est une application mobile permettant de se connecter à votre serveur Kavita personnel pour accéder à votre bibliothèque de mangas et comics. Cette politique de confidentialité explique comment vos données sont traitées.

2. Collecte de données

Inkavi ne collecte AUCUNE donnée personnelle.

  • Aucune donnée n'est envoyée à des serveurs tiers
  • Aucun système de tracking ou d'analyse n'est intégré
  • Aucune publicité n'est affichée

3. Stockage des données

Toutes les données sont stockées localement sur votre appareil :

  • URL du serveur Kavita : Stockée de manière sécurisée (Keychain iOS / Keystore Android)
  • Clé API : Stockée de manière sécurisée
  • Token JWT : Stocké de manière sécurisée, expire après 10 jours
  • Progression de lecture : Stockée localement et synchronisée avec VOTRE serveur Kavita uniquement
  • Préférences de l'application : Stockées localement

4. Communication avec votre serveur Kavita

Inkavi communique uniquement avec le serveur Kavita que vous configurez. Toutes les données échangées transitent directement entre votre appareil et votre serveur :

  • L'URL du serveur est fournie par vous
  • La connexion utilise le protocole HTTPS (si configuré sur votre serveur)
  • Aucun intermédiaire n'est impliqué

5. Autorisations de l'application

Inkavi demande les autorisations suivantes :

  • Internet : Pour se connecter à votre serveur Kavita
  • Stockage : Pour mettre en cache les images et améliorer les performances
  • Biométrie (optionnel) : Pour protéger l'accès à l'app par empreinte digitale ou Face ID

6. Sécurité

Nous prenons la sécurité au sérieux :

  • Utilisation de Keychain (iOS) et Keystore (Android) pour stocker les identifiants
  • Support du protocole HTTPS
  • Authentification par JWT avec expiration automatique
  • Code PIN optionnel pour protéger l'accès à l'application

7. Données des enfants

Inkavi ne collecte aucune donnée personnelle, y compris celle des enfants de moins de 13 ans. L'application est conçue pour être utilisée avec supervision parentale.

8. Modifications de cette politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité. Toute modification sera publiée sur cette page avec une nouvelle date de mise à jour.

9. Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter via notre dépôt GitHub : https://github.com/crashcool69/Inkavi
